System Requirements

TextWrangler 4.0 requires Mac OS X 10.6 or later (10.6.8, 10.7.3 or later recommended). It will not run on Mac OS 9, or any earlier versions of Mac OS X.

Feature Highlights

TextWrangler 4 offers numerous new features and interface enhancements, including:

  • Automatically preserves unsaved documents upon quit
  • Save and load custom text color schemes
  • Simplified preferences and configuration management
  • Direct editing of files within Zip archives—plus multi-file search & replace!
  • In-window Live Search to highlight and quickly jump between matches
  • Modeless Find and Multi-File Search windows provide a consistent, familiar interface to TextWrangler's potent search and replace capabilities

as well as all the powerful core features that TextWrangler is known for.

Info on New Features

In addition to these major enhancements, TextWrangler 4 also contains numerous additional behavioral and interface refinements, as well as performance enhancements and bug fixes. For a detailed summary of changes and bug fixes, please refer to the current release notes, which are available on our web site.
